# Data Science Projects

This repository contains a collection of my data science projects, demonstrating my skills and expertise in the field. Each project demonstrates different aspects of data analysis, machine learning, and visualization.

# Projects

1.  **[Customer Churn Prediction](./Customer-Churn)**
      - **Description:** This project predicts customer churn and identifies factors associated with customer attrition.
      - **Technologies Used:** Logistic Regression, K-Nearest Neighbors (KNN), Support Vector Machine (SVM), Decision Tree Classification
      - **Results:** The Decision Tree model performed best with 73.5% accuracy and an AUC-ROC score of 0.78.

2. **[Medical Diagnosis Prediction](./Medical-Diagnosis)**
      - **Description:** This project predicts the likelihood of developing a medical condition based on patient biomarkers and health characteristics.
      - **Technologies Used:** Logistic Regression, K-Nearest Neighbors (KNN), Support Vector Machine (SVM), Decision Tree Classification, 5-Fold Cross Validation
      - **Results:** The Decision Tree model performed best, achieving a recall score of 54.3% under 5-fold cross-validation.

3. **[Credit Card Fraud Detection](./Credit-Card-Fraud-Detection)**
      - **Description:** This project detects fraudulent transactions in highly imbalanced financial datasets.
      - **Technologies Used:** Logistic Regression, Support Vector Machine (SVM)
      - **Results:** Logistic Regression achieved the strongest performance with a recall score of 33.3% and an AUC-ROC of 0.47.

4. **[Customer Segmentation Using Hierarchical Clustering](./Customer-Segmentation)**
      - **Description:** This project uses agglomerative hierarchical clustering to uncover hidden customer segments within demographic and behavioral data.
      - **Technologies Used:** Agglomerative Hierarchical Clustering, Dendrogram Analysis
      - **Results:** Four distinct customer segments differentiated by age and income were identified.
